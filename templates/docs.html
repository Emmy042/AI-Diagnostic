Overview

This project is an AI-powered Dermatology Diagnostic Support system developed as an academic prototype. It is designed to assist in clinical decision-making by analyzing skin images and predicting potential dermatological conditions.

Architecture

The system utilizes a deep learning model based on the InceptionV3 architecture, fine-tuned on a dataset of dermatological images. The web interface is built using Flask, providing a lightweight and responsive prototype for uploading images and viewing results.

Features

  • Image Classification: Predicts the likelihood of various skin conditions.
  • Confidence Scoring: Provides a percentage-based confidence level for the prediction.
  • Clinical Notes: Generates relevant clinical observations based on the classification.
  • Referral Guidance: Suggests next steps or triage recommendations.
  • Demo Mode: A safe, fallback mode that simulates the diagnostic process without requiring a fully trained model.

Usage

To use the application, navigate to the home page, select a clear image of the skin condition (in JPG, PNG, or WEBP format), and click "Analyze Image". The system will process the image and display a detailed diagnostic report.
